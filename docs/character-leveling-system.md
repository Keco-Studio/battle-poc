# 角色等级系统设计文档

## 1. 角色属性

### 基础属性
| 属性 | 初始值 | 每级成长 |
|------|--------|----------|
| HP   | 100    | +30      |
| MP   | 50     | +15      |
| ATK  | 5      | +5       |
| DEF  | 3      | +3       |
| SPD  | 3      | +3       |

### 等级系统
- 初始等级: 1
- 每级所需经验: `当前等级 * 10`
- 升级公式: 1级需要10点经验，2级需要20点，3级需要30点...

### 升级效果
1. 回复满HP
2. 回复满MP
3. 各项属性按成长值增加
   - HP 每级 +30
   - MP 每级 +15
   - ATK 每级 +5
   - DEF 每级 +3
   - SPD 每级 +3
4. 可能解锁新技能

---

## 2. 经验系统

### 获取经验
- 战胜怪物获得: `怪物等级` 点经验
- 例: 战胜Lv.3怪物获得3点经验

### 经验显示
- 显示当前经验 / 升级所需经验
- 例: Lv.1 显示 `3/10`，升级后变为 `0/20`

---

## 3. 技能系统

### 技能等级
| 技能ID | 名称   | 解锁等级 | 伤害/效果 | 描述           |
|--------|--------|----------|-----------|----------------|
| 1      | 重击   | 1        | ATK*1.5   | 基础攻击技能   |
| 2      | 防御   | 1        | -         | 减少下次50%伤害|
| 3      | 连击   | 2        | ATK*0.8   | 两次攻击       |
| 4      | 治疗   | 3        | ATK*0.5   | 恢复HP         |
| 5      | 强击   | 5        | ATK*2.0   | 高伤害技能     |
| 6      | 反击   | 7        | ATK*1.2   | 反伤+攻击      |

### 技能效果计算
- 伤害技能: `ATK * 技能倍率`
- 治疗技能: `ATK * 倍率`

### 战斗技能栏
- 最多同时显示4个技能
- 根据当前等级显示已解锁技能
- 未解锁技能显示为锁定状态

## 6. 数据结构

```typescript
interface PlayerStats {
  level: number;
  exp: number;
  hp: number;
  maxHp: number;
  atk: number;
  def: number;
  spd: number;
}

interface Skill {
  id: number;
  name: string;
  unlockLevel: number;
  damageMultiplier: number; // ATK倍率的伤害，0表示特殊效果
  effect?: 'heal' | 'defense' | 'counter';
  description: string;
}
```

---

## 7. 升级流程

1. 战斗胜利 → 获得经验
2. 经验 >= 升级所需 → 自动升级
3. 升级效果:
   - `exp -= 升级所需经验`
   - `level++`
   - `maxHp += 10`
   - `hp = maxHp` (满血)
   - `atk += 5`
   - `def += 3`
   - `spd += 3`
4. 检查是否有新技能解锁
